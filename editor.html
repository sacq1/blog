如何发布文章到 GitHub

  1. 第一步：在这里写好文章 填写标题、分类、内容，点击「生成 HTML」按钮，复制生成的代码。
  2. 第二步：打开 GitHub 仓库 访问 github.com/sacq1/blog，进入仓库页面。
  3. 第三步：进入 posts 文件夹 点击 posts 文件夹，进入文章目录。
  4. 第四步：创建新文件 点击右上角「Add file」→「Create new file」。
  5. 第五步：命名文件 文件名输入 你的文章名.html（英文或拼音，如 my-first-climb.html）。
  6. 第六步：粘贴代码 将复制的HTML代码粘贴到文件内容区域。
  7. 第七步：提交文件 滚动到底部，点击绿色「Commit changes」按钮提交。
  8. 第八步：回到仓库根目录，更新文章列表 打开 js/main.js 文件，在 POSTS 数组中添加新文章信息。点击编辑按钮（铅笔图标），修改后提交。
提示：添加文章信息时，格式如下：
{ id: 4, title: '标题', excerpt: '摘要', date: '2026-05-10', category: '攀岩', emoji: '🧗', bg: 'linear-gradient(135deg, #2c1810, #1a1a2e)', url: 'posts/文件名.html', hidden: false }
