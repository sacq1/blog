如何在 GitHub 中发布文章

                    

                        

                            
1

                            

                                
打开 GitHub 仓库

                                
访问你的博客仓库：github.com/sacq1/blog（或你改过用户名后的地址）

                            

                        


                        

                            
2

                            

                                
进入 posts 文件夹

                                
在仓库页面点击 posts/ 文件夹进入。
首次需要自己创建这个文件夹：点击右上角 "Add file" → "Create new file"。

                            

                        


                        

                            
3

                            

                                
创建新文件

                                
点击右上角 "Add file" → "Create new file"。
在文件名输入框中输入：你的文章名.html（比如 my-first-climb.html）
注意文件名必须用英文/拼音，不要用中文。

                            

                        


                        

                            
4

                            

                                
粘贴 HTML 代码

                                
将上面生成的 HTML 代码全部复制（点击"复制代码"按钮），粘贴到文件编辑器中。

                            

                        


                        

                            
5

                            

                                
提交文件

                                
在页面底部填写提交信息（如"添加新文章：xxx"），选择 "Commit directly to the main branch"，点击 "Commit new file"。

                            

                        


                        

                            
6

                            

                                
更新文章列表

                                
回到仓库首页，点击 js/main.js 文件。
在 POSTS 数组中添加你的新文章信息。
格式如下（复制并修改）：

                                
{
    title: '你的文章标题',
    excerpt: '文章摘要（一句话）',
    date: '2026-05-10',
    category: '攀岩',
    emoji: '🧗',
    bg: 'linear-gradient(135deg, #2c1810, #1a1a2e)',
    url: 'posts/xxx.html'
},

                                
点击 "Commit changes" 保存。等待1-2分钟，网站自动更新。

                            

                        

                    

                


                
                

                    
如何删除或隐藏文章

                    

                        

                            
1

                            

                                
方法一：隐藏文章（推荐）

                                
在上方"文章管理"区域中，找到你要隐藏的文章，设置 "hidden":true，然后点击"保存管理"。
文章文件还在，但不会在首页和文章列表中显示。

                            

                        

                        

                            
2

                            

                                
方法二：完全删除

                                
进入 GitHub 仓库 → posts/ 文件夹 → 找到要删除的文章文件 → 点击右侧"..." → "Delete file" → Commit changes。
同时在 js/main.js 的 POSTS 数组中删除对应项。
